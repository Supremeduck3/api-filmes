# 🎬 API de Filmes
> API RESTful para gerenciamento de filmes com Node.js, Express e Prisma.

![Node.js](https://img.shields.io/badge/Node.js-43853D?style=for-the-badge&logo=node.js&logoColor=white)
![Prisma](https://img.shields.io/badge/Prisma-3982CE?style=for-the-badge&logo=Prisma&logoColor=white)

## 📌 Endpoints: GET POST PUT DELETE /filmes/:id

## 🚀 Como Rodar
```bash
git clone https://github.com/Supremeduck3/api-filmes.git
cd api-filmes && npm install && npx prisma migrate dev && npm run dev
```
